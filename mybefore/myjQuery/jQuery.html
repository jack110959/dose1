# jQuery

# 一，原理

采用闭包，在jq的js文件中采用匿名函数自调用，将一些方法添加到windon对象中。

在js文件中也使用了windon.jquery=windon.$=jquery

引入

设置crossorigin="anonymous" 可以在请求资源时不带cookie。

<script crossorigin="anonymous" src="https://cdn.bootcdn.net/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

# 三，操作属性

需要注意：上面选择器返回的都是数组。所以不能直接使用**.属性**获取属性值

获取属性值

attr("type")：获取对应属性值 ------->底层对应getAttribute("abc")方法，只能获取初始值。

val()：获取value值--------->底层相当于对象.value,获取的都是当前值。

<input class="com" type="button" value="测试" onclick="testsx()"><br>
<input type="text" name="" id="inpid" value="">

<script>
   function testsx(){
       var inp=$("#inpid");
       alert(inp.attr("type")+"=="+inp.attr("value")+"=="+inp.val())
   }
</script>

修改属性值

attr("type","button");----->底层对应setAttribute("abc","呵呵");

function test111(){
       var inp=$("#inpid");
       inp.attr("type","button");
}

# 五，操作css

单个css属性的操作

<div id="div1" onmouseover="changecss()">
        <b>默认值</b>
</div>

<script>
 function changecss(){
    var div=$("#div1");
    alert(div.css("width"))
    div.css("width","100px")
    div.css("border","solid 1px red")
    div.css("height","100px") 
 }
</script>

多个css属性的操作

function changecss2(){
    var div=$("#div1");
    div.css({"width":"100px","height":"100px","border":"solid 1px red"})
}

修改class值

<div id="div1" onmouseover="changeclass()" onmouseout="changeclass2()">
        <b>默认值</b>
</div>

<script>
    function changeclass2() {
        var div = $("#div1");
        div.removeClass("com");//移除class值
    }

    function changeclass() {
        var div = $("#div1");
        div.addClass("com");//追加class值
    }
</script>
<style>
.com {
    background-color: orange;
}
</style>

# 六，操作文档结构

内部添加

  • append("新元素")：在对象元素的后面加上新元素
  • appendTo("元素对象"):将某个元素对象加到另一个元素中(后面添加)
  • prepend("新元素"):在对象元素前加上该新元素
  • prependTo():将某个元素对象加到另一个元素中（前面添加）

外部添加

after()：外部插入，插到指定元素后面 before():外部插入：插到指定元素前面 insertAfter()与insertBefor() ：它们两个与prependTo和appendTo()类似，只不过是外部插到后面和前面

包裹：

wrap()具体用法查询api a.wrap(content)，将a对象外包裹指定的content元素

替换：

replaceWith和replaceAll()把字句与被字句 a.replaceWith(content) 使用content将a对象替换，包括标签内容

删除： 建议使用

empty() ：删除标签下的所有子节点

复制：

clone() a.clone() 复制一份a 通常和append()连用:a.clone().appendTo(b)

# 七，操作事件

需要注意：

js实现添加事件为覆盖效果，也就是添加多次都会进行覆盖，最终按钮都只有一次事件

jq实现添加事件为追加效果，也就是添加多次都会进行叠加，最终按钮为叠加的总数。

js实现添加事件

<input type="button" onclick="test()" value="测试">
<input type="button" id="btn" value="效果按钮"><br>

<script>
    function test(){
        var btn=document.getElementById("btn");
        btn.onclick=function (){
            alert("js实现");
        }
    }
</script>

jq实现添加事件

<input type="button" onclick="test2()" value="测试">
<input type="button" onclick="test3()" value="删除">
<input type="button" id="btn2" value="效果按钮"><br>

<script>
  function test2(){
        $("#btn2").bind("click",function(){alert("jq实现")})
    }
   function test3(){
       //删除通过bind()方法添加的事件()
        $("#btn2").unbind("click");
    }
</script>

其他事件添加

one方法，添加只有一次效果的事件

ready方法，页面载入事件(能够多次添加，不会被覆盖)

 $(document).ready(function (){
     alert("页面载入事件")
 })

# 九，基本使用

在页面载入事件的方法中，为需要事件的元素添加事件，从而达到将jq的js代码隔离出来。

//载入页面事件
$(function(){
    //添加事件
    $("#dh").bind("click",function(){
        alert("页面载入时为元素添加的事件");
    })
})
