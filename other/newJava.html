# java8新特性

主要用来写一些java新特性如何去使用

# 一，Lambda表达式

这个主要为只有一个抽象方法的接口实例化的一种表达形式。

这种只具有一个抽象方法的接口也被称为函数式接口。

这里就简单举几个例子，能够看懂就好。

注意记得 -> 符号左边为抽象方法的形参，右边为方法内部的操作。

先看一下原来的写法

Comparator<Integer> com3 = new Comparator<Integer>() {
			@Override
			public int compare(Integer o1, Integer o2) {
				// TODO Auto-generated method stub
				return 0;
			}
		};

下面就是简化之后的lambda表达式写法

Comparator<Integer> com1 = (o1, o2) -> Integer.compare(o1, o2);
		System.out.println(com1.compare(90, 2));
      /*下面为方法的引用，一般情况下
      当方法体中的对象或类的某个方法的形参跟返回值类型
      跟要实现的抽象方法一致时可以使用 */
      /* 方法体中的Integer类的compare方法的形参与返回值跟
         Comparator接口中的抽象方法一致*/
       /* 基本的三种类型
          对象::实例方法
          类::静态方法
          类::实例方法
       */
		Comparator<Integer> com2 = Integer::compare;
		System.out.println(com2.compare(29, 90));

如果自己想使用的画采用最基本的写法就可以了

Comparator<Integer> com6 = (o1, o2) -> {
			/* 其他操作 */
			return Integer.compare(o1, o2);};
		System.out.println(com6.compare(29, 90));

# 二，函数式接口

# 1，Consumer接口

当你希望在不同的场景下希望某个对象的方法能够执行不同的操作时可以使用。

简单举例如下

public void show(Double money,Consumer<Double> con) {
		con.accept(money);
	}
	
	@Test
	public void test1() {
		/* 原来的写法 */
		show(200.00,new Consumer<Double>() {
			@Override
			public void accept(Double t) {
				// TODO Auto-generated method stub
				System.out.println("money="+t);
			}
		});
        
        /* 也可以使用lambda表达式 */
		show(500.00,t->{System.out.println("money="+t);});
        
	}

# 2，Predicate接口

当你从数据库中得到数据，你希望能够根据某些条件过滤这些数据。可以使用。

该接口能够让你在使用的时候，再自己编写过滤规则。

简单举例如下

这里以List集合为例，因为从数据库中取出数据一般都存放于List集合中。

public List <String> filetList(List <String> list,Predicate<String> pre) {
		List<String> newlist=new ArrayList<String>();
		
		for (String string : list) {
			if(pre.test(string)) {
				newlist.add(string);
			}
		}
		return newlist;
	}
	@Test
	public void test2() {
		List<String> list=Arrays.asList("北京","天津","南京","武汉");
		
		/* 普通写法 */
		List<String> newList1=filetList(list,new Predicate<String>() {
			@Override
			public boolean test(String t) {
				// TODO Auto-generated method stub
				return t.contains("京");
			}
		});
		System.out.println(newList1);
		
		/* 使用lambda表达式 */
		List<String> newList2=filetList(list,t-> {
			return t.contains("京");
		});
		System.out.println(newList2);
		
		
	}

最后就是在开发中如何使用，可以将该方法定义在工具类中，在使用时实现test方法重新定义过滤规则即可。

public static <T> List <T> filetList(List <T> list,Predicate<T> pre) {
		List<T> newlist=new ArrayList<T>();
		for (T t : list) {
			if(pre.test(t)) {
				newlist.add(t);
			}
		}
		return newlist;
	}

# 三，Stream的使用

Stream是Java8的一大亮点，是对容器对象功能的增强，它专注于对容器对象进行各种非常便利、高效的 **聚合操作（aggregate operation）**或者大批量数据操作。Stream API借助于同样新出现的Lambda表达式，极大的提高编程效率和程序可读性。同时，它提供串行和并行两种模式进行汇聚操作，并发模式能够充分利用多核处理器的优势，使用fork/join并行方式来拆分任务和加速处理过程。所以说，Java8中首次出现的 java.util.stream是一个函数式语言+多核时代综合影响的产物。

# 1，得到按某种规律生成的数据-iterate()

如果你想得到某种自定义规律的数据时可以使用下面的操作

List<Integer> list=new ArrayList<Integer>() ;
	Stream.iterate(0,t->t+3).limit(10).forEach(t->{list.add(t);});
	System.out.println(list);

iterate方法，第一个参数为种子，表示开始的数据，第二为自定义规则，limit为限制取得多少个数据，forEach为终止操作，可以在里面为生成的数据进行操作。

# 2，过滤某些数据-filter()

如果你从数据库得到的数据不符合要求，需要过滤掉某些数据时可以使用如下方法。

@Test
public void test4() {
	List<Person> list=getList();
	Stream<Person> stream = list.stream();
	List list1=new ArrayList<Person>();
	stream.filter(e->e.getAge()<1004).forEach(t->{list1.add(t);});
	System.out.println(list1); 
}

# 3，可以用于分页获取数据

两个方法，skip(long n)和limit(long n)

skip(long n)用来跳过多少个数据

limit(long n)用来限制获取多少个数据

简单举例如下，跳过两个数据后再获取接下来的3个数据

@Test
public void test5() {
    /* 从数据库获取的list数据 */
	List<Person> list=getList();
	Stream<Person> stream = list.stream();
    /* 用来存放想要的数据 */
	List list1=new ArrayList<Person>();
	stream.skip(2).limit(3).forEach(t->{list1.add(t);});
	System.out.println(list1); 
}

如果想用来做分页，可以用以下的方法，该方法主要是在后端就完成了分页的所有操作，不过每一次都需要重新查找数据库，效率不高。

public static <T> List<T> rList(List<T> list, Integer pagenum, Integer size) {
		Integer total=(pagenum-1) * size;
		/* 获取总页数 */
		Integer totaopageNum=list.size()%size;
		if(totaopageNum==0) {totaopageNum=list.size()/size-1;}
		else {totaopageNum=list.size()/size;}
		
		/* 如果超过总页数了就返回最后一页的数据 */
		if(total>list.size()) {total=totaopageNum*size;}
		
		List newlist = new ArrayList<Person>();
		list.stream().skip(total).limit(size).forEach(t -> {
			newlist.add(t);
		});
		return newlist;
	}

# 4，去掉重复的数据

如果List存的是对象时，需要重写该对象的hashcode和equal方法，这样判断是否重复的依据才是对象里面的值，而不是地址值。

@Test
	public void test7() {
		/* 从数据库获取的list数据 */
		List<Person> list = getList();
		Stream<Person> stream = list.stream();
		List<Person>list1 = new ArrayList<Person>();
		stream.distinct().forEach(t -> {
			list1.add(t);
		});
	}

# 5，获取对象单个属性的数据-map()

map(Function f)，接收一个函数作为参数，用来将元素转换为其他形式或提取信息。

flatMap(mapper)，能够将其他Steam对象的中数据一个一个的映射出来。而map会将整个Stream对象作为一个数据。

有时候你只是想获取数据库表中某个属性的数据，并希望能够过滤掉一些不符合规则了。可以使用。

@Test
	public void test8() {
		/* 从数据库获取的list数据 */
		List<Person> list = getList();
	
		Stream<Person> stream = list.stream();
        /* 映射为只有Name属性的Stream */
		Stream<String> mapName = stream.map(Person::getName);

		List<String>list1 = new ArrayList<String>();
		
		mapName.filter(e->e.length()>2).forEach(t -> {
			list1.add(t);
		});
	}

# 6，排序

sorted方法中，如果没有使用定制排序的Comparator对象作为参数就会使用对象中的自然排序。

@Test
	public void test9() {
		/* 从数据库获取的list数据 */
		List<Person> list = getList();
		list.stream().sorted((o1,o2)->{
			int agevalue=Integer.compare(o1.getAge(), o2.getAge());
			if(agevalue!=0) {return agevalue;}
			else return o1.getName().compareTo(o2.getName());
		}).forEach(System.out::println);
	}

# 7，统计

# (1)统计个数

如果你想知道对象中的属性符合某个规则的个数有多少。

比如年龄大于多少的有多少个人，姓名含有什么字的有多少个人等。

简单举例如下

@Test 
	public void test10() {
		List<Person> list=getList();
		long count = list.stream().filter(t->t.getAge()>1004).count();
		System.out.println(count);
	}

# (2)统计最大值或最小值

如果你想得到某个属性的最大值，可以使用如下方法，

需要注意的是该方法只能返回一个数值，所以你只能得到最大数值为多少，无法得到属性为最大值的所有对象。

@Test 
public void test10() {
	List<Person> list=getList();
	Optional<Integer> max =list.stream().map(Person::getAge).max(Integer::compare);
	System.out.println(max.get());
	}

# 8，规约

reduce（）可以统计某个属性的总数。

比如你想获取所有人的余额总额，或某个月的总销售量等。

可以使用如下操作

@Test
	public void test11() {
		List<Person> list = getList();
		Integer reduce = list.stream().map(Person::getAge).reduce(0, Integer::sum);
		System.out.println(reduce);
	}

# 9，收集

collect（）可以用来将stream中进行各种操作后的数据返回成一个集合

返回List，如下

@Test
	public void test12() {
		List<Person> list = getList();
		List<Person> collect = list.stream().filter(t-                                         >t.getName().contains("京")).collect(Collectors.toList());
		collect.forEach(System.out::println);
	}

返回set如下

@Test
	public void test13() {
		List<Person> list = getList();
		Set<Person> collect = list.stream().filter
        (t->t.getAge()>1003).collect(Collectors.toSet());
		collect.forEach(System.out::println);
	}
